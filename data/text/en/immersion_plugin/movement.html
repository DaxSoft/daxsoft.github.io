Description

You will be able to set different animations for the npc/player characters. All configuration will be stored into a .json file. For example: You can set up a different animation for when the player is idle, in which: - You can change the sprite characterset (picture) - The speed - Frame position and so on There will be a editor to edit it more easy :)

Features

You shall set up a collision body for each npc/player. It will be a default setup. However it's recommended to costum for each one. - You can set up different shapes of collision body (circle, rectangle, polygon) - You can set up the position in which the collision body will be (on the lower-body of the player and so on) - You can set up different shapes of collision body for each animation state

You can create several states of animations for the character, such as: - Idle Animation for when the character isn't moving. - Dash Animation for when the caracter is running - Each states has his own properties values, such to control the frame speed.
