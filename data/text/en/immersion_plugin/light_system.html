Description

For this project I have adapted the 'pixi-lights' to let you create a gorgeous ambience for your game. So, you will be able to create lights using this system as well, you can create 'normal lights', in which is using normal pictures (sprites) with blend modes. To work correctly, every sprite needs to have a normal-map texture. The file, needs to be on the same folder that the diffuse sprite (file), with the prefix '!' followed by the name of the diffuse name: 'SM_uVendedor.png | !â€‹SM_uVendedor.png'

Features

You are able to create lights using 'pixi-lights', in which includes: * ambient-light * directional-light * point-light All the variables such position, lightHeight and so on, can be edited on the editor.

You are able to create lights using PIXI.Sprite, in which means, using pictures as light source. To create the 'sense of light', you can change the blend modes at will.

The system has a complete color system, in which includes: * Possibility to change the RGB values * List of defined colors to use at will It will be included: * jfsiii/chromath to handle with colors.

You can add pixi-filters on lights. However, only on sprite-lights. Such, for instance, BlurFilter

You can set up conditions to, only when the conditions meets true, the light source will be visible. * Time Period : only when the period meets (such, only in night) * Switch ID : only when the gameSwitch from ID x, meets true. * Floor : only when the player is on the floor X

Pulse effect on lights, useful for creating 'explosion light effect', or whatever

Oscilation effect on lights, useful for creating lights with some 'energy problems', like lamps on dungeons and whatever.
