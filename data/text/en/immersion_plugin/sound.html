Description

You will be able to create 3D spatial sounds, to give a immersive feeling for your game.

Features

You can set up conditions to, only when the conditions meets true, the sound source will be visible. * Time Period : only when the period meets (such, only in night) * Switch ID : only when the gameSwitch from ID x, meets true. * Floor : only when the player is on the floor X

You will can set up a collision shape to: when the player is inside, for each pace that the player takes, it will emmits the sound, based on the position of his feets. It's like when the player is on a grass field, and for each step that the player takes, the sound of grass being stepped on.
