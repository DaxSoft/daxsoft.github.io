Descrição

Você poderá criar grandes mapas sem a limitação do editor padrão da engine. Então, para cada mapa, você pode definir três andares (under, base, high) E, para cada andar, você pode definir 5 camadas (background, ground, npc/objects, above npc/objects, above all). Como também, para cada andar, você poderá definir diferentes luzes, partículas e por aí.

Features

Para cada mapa, você pode definir três andares. Claro, se quiser, só um. Por padrão, é 'base' Para cada andar, você poderá definir seus elements únicos.

Eu adaptei o 'davidfig-pixi-viewport', para criar um efeito legal na camera. Então, a câmera irá seguir em uma maneira suave, para mostrar o mapa. - Você pode definir o alvo da camera. - Você pode moficiar a velocidade da camera. - Você pode definir efeitos de zoom
