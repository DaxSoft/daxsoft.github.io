Descrição

Você será capaz de criar sons espaciais, para dar um sentimento imersivo para seu jogo.

Features

Você pode definir condições para, quando somente as condições forem aceitas, o som será validada. - Perído do dia : somente quando, por exemplo, estiver de noite - floor : somente quando o jogador estiver no mesmo andar - switch : somente quando o valor da Id X de uma switch do jogo estiver verdadeiro.

Você poderá definir uma colisão, na qual quando o jogador estiver dentro, para cada passo que ele der, emitirá um som. Como quando o jogador está num campo de grama, e cada passo que ele der, o som da grama sendo pisado.
