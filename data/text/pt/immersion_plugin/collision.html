Descrição

Para esse projeto, eu adaptei o 'sinova-collision' para criar o sistema de colisão. Como também, eu criei um sistema de movimento por pixel que segue a grid de 8x8.

Features

Corpo de colisão pré-definido, no formato de círculo, poderá editar: - radius - position - average scale - padding

Corpo de colisão pré-definido, no formato de retângulo, poderá editar: - position - scale (x, y) - angle - padding - average scale

Corpo de colisão pré-definido, no formato de polígonos, poderá editar: - scale (x, y) - average scale - position - angle - add points - delete points - padding

Você pode definir condições para, quando somente as condições forem aceitas, a colisão será validada. - floor : somente quando o jogador estiver no mesmo andar - switch : somente quando o valor da Id X de uma switch do jogo estiver verdadeiro.

Para o jogador entrar em outro andar do mapa, é necessário uma colisão com essa propriedade. * horizontal - para quando a passage está na direção cima/baixo * vertical - para quando a passage está na direção esquerda/direita
