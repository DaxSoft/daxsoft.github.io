Descrição

Você irá poder usar o editor de partículas online, para criar lindas partículas para seu jogo. Usando o 'pixi-particles'.

Features

Você pode definir condições para, quando somente as condições forem aceitas, a partícula será validada. * Perído do Tempo : quando, por exemplo, é noite * floor : somente quando o jogador estiver no mesmo andar * switch : somente quando o valor da Id X de uma switch do jogo estiver verdadeiro.
