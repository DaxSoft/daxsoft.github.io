Descrição

Você poderá definir diferentes animações para cada npc/plater. Toda configuração será salvada em um arquivo .json. Por exemplo: Você pode definir uma animação diferente para quando o jogador estiver parado: - Mudando a imagem - A velocidade - Posição do frame Haverá um editor para facilitar

Features

Você deverá definir um corpo de colisão para cada npc/player. Haverá uma configuração padrão, contudo eu recomendo costumizar. - Você pode definir formatos de colisões como circular, retângulo - Você pode definir a posição na qual o corpo de colisão irá estar, como nos pés do jogador. - Você pode definir diferentes corpos de colisão para cada animação;

Você pode criar vários estados de animação para o personagem, como: - Animação Parada para quando o personagem não estiver movento. - Animação Correndo - Para cada estado de animação, você poderá costumizar as propriedades.
