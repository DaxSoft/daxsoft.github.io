Features

Para esse projeto eu adaptei o 'pixi-lights' para deixar você criar uma ambientação incrível pro seu jogo. Então, você será capaz de criar luzes usando esse sistema, como também, criar luzes 'normais', na qual é usando imagens (sprites) com blend modes.

Para esse projeto eu adaptei o 'sinova-collision' para criar o sistema de Colisão. Como também, eu criei um sistema de 'movimento por pixel', que segue a grid de 8x8. Contudo, sprites e objetos podem mover livremente, a grid será usada apenas como referência. Para os movimentos de animações e corpos de colisões para os sprites, todas as configurações serão salvas em um arquivo .json, e terá um editor para facilitar.

Sistema de tempo padrão, que irá permitir você criar momentos e ambientação específicos. Você pode, por exemplo, criar luzes e comportamentos para NPCs que serão ativados somente quando, o período do dia é noite.

Para esse projeto, eu adaptei o 'pixi-particles', então você poderá usar o editor online disponível. Haverá também um editor para você configurar.

Eu irei criar um sistema de clima que combina especificamente com games 2D. O conceito é usando pontos & terrenos como emissores.

Para esse projeto eu adaptei o 'goldfire-howler' para permitir você a criar ambientação única de sons. Na qual, você poderá criar sons usando posições para influênciar o volume e por aí.

Nesse projeto não haverá usos no editor padrão de mapas do RPG Maker MV, salvo para configurar eventos. Com esse plugin, você poderá criar mapas no Photoshop (recomendo esse), ou no Gimp, ou tanto faz o programa, desde que você siga as regras do sistema... No mapa, você pode ter três andares (baixo, base, alto). Em cada andar, você pode ter 5 camadas (background, chão, npc/objetos, acima de npc/objetos, acima de tudo). E, para cada andar, você poderá definir diferentes fontes de luzes, partículas, sons e por aí. Em si, são três mapas em um só.

Há um editor de mapas na qual você poderá em jogo, para ao mesmo tempo que você edita, você notar como a performance está sendo afetada. Incluí: - Editor de luzes - Editor de partículas - Editor de clima - Editor de colisão - Editor de filtros - Editor de sprites
