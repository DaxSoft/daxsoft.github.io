Descrição

Para esse projeto eu adaptei o 'pixi-lights' para que você possa criar uma ambientação incrível. Então, você poderá criar luzes usando esse sistema, como também, criar utilizando imagens, com blend modes. Para funcionar corretamente, cada sprite precisará ter uma textura de 'normal-map'. O arquivo, deverá estar na mesma pasta que o a textura 'diffuse', com o prefixo '!', seguido pelo mesmo nome do arquivo da textura 'diffuse': 'SM_uVendedor.png | !​SM_uVendedor.png'

Features

Você será capaz de criar luzes usando 'pixi-lights', que incluí: * ambient-light * directional-light * point-light Todas as principais variáveis são editáveis no editor.

Você pode criar utilizando imagens, com blend modes.

O sistema tem um sistema de cores completo, na qual incluí: * Possibilidade mudar os valores RGB * Lista com cores já definidas Será incluso: * jfsiii/chromath.

Você poderá adicionar filtros nas luzes, contudo, somente nas 'sprite-lights'. Como, por exemplo, BlurFilter

Você pode definir condições para, quando somente as condições forem aceitas, a luz será validada. - Perído do dia : somente quando, por exemplo, estiver de noite - floor : somente quando o jogador estiver no mesmo andar - switch : somente quando o valor da Id X de uma switch do jogo estiver verdadeiro.

Efeito de pulsar nas luzes, útil para criar algo como 'efeito de explosão'

Efeito de oscilação nas luzes, útil para criar luzes com problemas de energia
